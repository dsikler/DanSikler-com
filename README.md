# Dansikler.com

A modern, professional portfolio website showcasing ventures in tech, real estate, and e-commerce.

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Lucide React** for icons

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
app/
  ├── about/          # About Me page
  ├── resume/         # Resume & Skills page
  ├── budget-app/     # Budget App page
  ├── elevate/        # Elevate Asset Management LLC page
  ├── yak-bones/      # Yak Bones e-commerce page
  ├── layout.tsx      # Root layout with Header and Footer
  ├── page.tsx        # Homepage
  └── globals.css     # Global styles

components/
  ├── Header.tsx      # Sticky navigation header
  ├── Footer.tsx      # Footer with social links
  ├── Hero.tsx        # Homepage hero section
  └── VentureCard.tsx # Reusable venture card component
```

## Design System

- **Primary Color**: Navy blue (#1e3a8a)
- **Accent Color**: Teal (#0d9488)
- **Typography**: Inter (Google Fonts)
- **Responsive**: Mobile-first design

## Pages

- **Home**: Hero section with 4 main venture cards
- **About**: Personal background and story
- **Resume**: Professional skills and accomplishments
- **Budget App**: Personal finance management tool
- **Elevate**: Real estate asset management company
- **Yak Bones**: E-commerce platform (coming soon)
